# Airbnb-Seattle-Listings-Analysis-Tableau-Dashboard
Interactive Tableau dashboard analyzing Airbnb listings in Seattle (2016 dataset). Explores pricing trends, neighborhood insights, property types, host performance, review scores, availability patterns, and key factors affecting rental performance in Queen Anne and across Seattle.

## 🎯 Project Overview

This project provides deep insights into Seattle's Airbnb ecosystem with a focus on:
- Pricing distribution and factors affecting price
- Neighborhood performance (especially Queen Anne)
- Property types and room types analysis
- Host characteristics and superhost performance
- Review scores and guest satisfaction
- Availability and occupancy trends
- Amenities impact on pricing

## 📊 Dashboard Features

- **Overview Dashboard**: Key metrics, price distribution, and market summary
- **Neighborhood Analysis**: Performance comparison across Seattle neighborhoods
- **Pricing Insights**: What drives higher/lower nightly rates
- **Host & Review Analysis**: Superhost impact, review scores breakdown
- **Availability & Calendar**: Booking patterns and occupancy trends
- **Property Deep Dive**: Room types, amenities, and capacity analysis

## 🛠️ Tools & Technologies

- **Tableau Desktop/Public** - Main visualization tool
- **Data Source**: Inside Airbnb Seattle dataset (January 2016)
- **Data Cleaning & Preparation**: Tableau Prep / Excel
- **File**: `Tableau Full Project.xlsx` (Listings + Calendar data)
